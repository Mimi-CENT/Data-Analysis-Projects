# Data Analysis Projects

This repository contains a collection of my data analysis projects using SQL and Excel. The projects focus on data cleaning, transformation, querying, and generating insights from structured datasets.

## Projects

### 1. Food Ordering Analysis (SQL)
- Designed a relational database with tables for customers, orders, and menu items
- Queried data using SQL to analyse order trends and item performance
- Applied aggregations and grouping to extract insights

### 2. Student Performance Analysis (SQL)
- Analysed student data using SQL
- Used aggregate functions (AVG, COUNT, MIN, MAX) to evaluate performance trends
- Queried datasets to compare results across subjects

### 3. Workforce Data Analysis (Excel)
- Cleaned and standardised inconsistent data fields
- Used XLOOKUP to combine datasets and map salary information
- Created pivot tables and charts to analyse diversity and pay distribution
- Presented findings using clear and accessible visualisations

## Tools Used
- SQL
- Excel
- DBeaver

## Notes
These projects were completed as part of my learning and practice in data analysis, with a focus on building strong foundational skills.
